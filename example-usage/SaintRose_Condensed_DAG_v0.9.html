Condensed defined-terms DAG

v0.9

Sample document used: 2019 Plan Document.pdf. This is the condensation graph of the defined-term dependency network. Every strongly connected component is collapsed into one super-node. The layout is a Sugiyama layered DAG with roots on the left and more composite structures to the right. Super-node size is proportional to total in-flow from the original definition graph.

This view exposes the semantic architecture more explicitly than the force layout: primitive inputs sit upstream, the mutual-definition core is condensed, and downstream composites sit to the right.
